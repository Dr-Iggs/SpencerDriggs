Overview

Several upper-level education courses include scoring students on their effectiveness as a teacher. They get graded on 22 categories, each by a 4-pt scale. Questions include:

  • Design student assessments
  • Communicating with students
  • Organizing physical space

These scores go back to 2020, and cover 35 different classes from 103 different teachers, for a total of 9854 observations from 4049 students.

Challenges

There are some challenges as we try to put Danielson scores into our final model of student success:

  • Students can get graded multiple times per class (Formative, Midterm, Final). However, not all students have all 3, and some students don’t even have a Final score in our dataset.
  • Students get graded on multiple classes through their time.
  • Teachers in those classes have their own opinions on how to score, and give different average scores through their history.

How to Consolidate

We want to use Danielson score both as a measure we can predict through early school experience, and as a predictor of post-college teaching success. To do this, we have to summarise these scores into a single value for each student.

  • Weighing important Danielson Questions
  • Adjusting scores for evaluator & class preferences
  • Multiple Scores per Class
  • Multiple classes

Both Idaho and BYU-I’s scoring systems gives equal weight to each question, but that doesn’t necessarily mean each question is equally important for teachers. We’ve re-weighted each of the questions based on personal preference, and we’ll see if this new version is a better predictor than the equal weighting used in classes.

A student that scores a 50 from a teacher who gives lots of 40s means much more than a teacher who gives 60s. We can replace the scores themselves with a z-score that puts each score within that context. We grouped by Teacher & Semester because some teachers gave wildly different average grades for the different classes they taught.

zscore=xStudentScore−μTeacherAndSemesterσTeacherAndSemester

Code 
z_danielson <- danielson %>%
  group_by(StudentID, Course) %>%
    slice(1) %>% #Grab only the most important score they have in the class
    ungroup() %>%
    mutate(overallSD = sd(total),
           WoverallSD = sd(weighted_total)) %>%
    group_by(EvaluatorCode,Course) %>%
    mutate(EvalAvg = mean(total,na.rm=TRUE),
           EvalSD = sd(total,na.rm=TRUE),
           EvalSD = replace_na(EvalSD,min(overallSD)),
           EvalSD = ifelse(n()<=5,overallSD,EvalSD), 
           WEvalAvg = mean(weighted_total,na.rm=TRUE),
           WEvalSD = sd(weighted_total,na.rm=TRUE),
           WEvalSD = replace_na(WEvalSD,min(overallSD)),
           WEvalSD = ifelse(n()<=5,WoverallSD,WEvalSD)
    ) %>% #When only one teacher, use the SD of all data
    ungroup() %>%
    mutate(z_score = round(ifelse(EvalSD==0,0,(total-EvalAvg)/EvalSD),3),
           weighted_z_score = round(ifelse(WEvalSD==0,0,(weighted_total-WEvalAvg)/WEvalSD),3),
           WDifference=weighted_z_score-z_score) #Catch a class of identical scores

Building Potential Y-Variables

But, why stop there? There are other ways we could represent each student’s Danielson performance, and some might be particularly useful.

  • We could see their range of scores, and how much they improved
  • We can keep just their ED 492 scores, for a perfect comparison between students
  • We can take an average of all their scores, not just the one from the most important class
  • We might calculate these on the unweighted scores as well
Code 
just_492s <- z_danielson %>%
    filter(Course=='ED 492') %>%
    mutate(ED492 = z_score,
           W_ED492 = weighted_z_score) %>%
    select(StudentID,ED492,W_ED492)
  
other_top_classes <- z_danielson %>%
  mutate(ranked_class = case_when(
    #492, any 400-level, 346, any 300-level
    str_detect(Course,'492') ~ '1',
    str_detect(Course,'4\\d\\d') ~ '2',
    str_detect(Course,'346') ~ '3',
    str_detect(Course,'3\\d\\d') ~ '4',
    TRUE ~ '5')) %>%
  arrange(StudentID,ranked_class) %>%
  slice(1, .by=StudentID) %>%
  summarise(LastClass = mean(z_score,na.rm=T),
            W_LastClass = mean(weighted_z_score,na.rm=T),.by=StudentID)

other_top_classes %>%
  left_join(just_492s,by='StudentID') %>%
  left_join(z_danielson %>%
              summarise(W_Mean_Zscore=mean(z_score,na.rm=T),
                        W_Mean_Zscore = mean(weighted_z_score,na.rm=T), .by=StudentID)
            ,by='StudentID') %>%
  left_join(z_danielson %>%
              summarise(Improvement = round(max(z_score)-min(z_score),3),
                        W_Improvement = round( max(weighted_z_score)-min(weighted_z_score),3),.by=StudentID),
            by='StudentID') %>%
  sample_n(5) %>%
  datatable(options = list(mode = 'multiple', selected = NULL, target = 'row', selectable = NULL))
